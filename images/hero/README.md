# Hero / Banner Images

Place full-bleed hero photos here.

Recommended:
- **Format:** JPG, minimum 1920×1200
- **Aspect:** landscape with meaningful space on one side for text overlay
- **File naming:**
  - `hero-home.jpg` (used on `index.html`)

The hero has a directional scrim + grayscale/contrast filter applied — photos that are a little flat still look great with the treatment baked in.
