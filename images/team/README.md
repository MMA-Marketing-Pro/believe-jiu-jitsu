# Team Photos

Place instructor and staff headshots here.

Recommended:
- **Format:** JPG or WebP
- **Crop:** Square or 4:5 portrait, minimum 800×800
- **File naming (kebab-case):**
  - `randy-gonzales.jpg` (head instructor)
  - `flavia-gonzales.jpg` (co-owner)

After dropping in a file, open the HTML and update the matching `<img src="images/team/...">` path — or use the `data-local` attribute already written into each `<img>` tag as a hint.
