# Gallery & Program Photos

Place real academy photos here — rolling on the mat, drilling, kids' classes, team photos.

Recommended:
- **Format:** JPG or WebP, minimum 1200px wide
- **File naming (kebab-case):**
  - `adults-teens.jpg`
  - `big-kids.jpg`
  - `little-kids.jpg`
  - `academy-mat.jpg`
  - `open-mat.jpg`
  - `competition.jpg`

These replace the Picsum placeholders used in `index.html` and `about.html` program cards. Update the `src` attributes (or use the `data-local` attribute on each `<img>` as a hint).
