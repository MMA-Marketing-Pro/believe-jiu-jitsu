> Ship Month 6. This is the "finish + polish" month.

## Prerequisites

Phases 1-5 live. You should now have ~134 programmatic pages live + the 7 original core pages = **141 pages** before Phase 6. This phase pushes total to ~164 and re-optimizes the earliest pages that are now 5-6 months old.

## Objective

Close the matrix + refresh early pages with new data.

- **20 new pages** — fill matrix gaps, cross-service × neighborhood combos not yet built, additional glossary or comparison pages based on 5 months of GSC data
- **10 refreshes** — take the oldest Phase 1 pages and update them with new testimonials, added sections, better photos, and any missing schema

**Total:** 30 pages (20 new + 10 refreshes).

## Page Selection Method

**Before building anything:** pull the following GSC and GA4 data:

1. Top 20 **impressions, low click-through** pages → these need better titles/meta/H1s
2. Top 20 **high-impression keywords NOT yet targeted** by a page → build those pages
3. Top 5 **pages with traffic but no conversions** → refresh with better CTAs and local proof
4. Bottom 10 **lowest-impression pages** → deeper unique content, more local signals, interlinking

Make the 20 new pages data-driven from real search performance — don't just build what was on the original matrix if the data tells a different story.

## Optimization Tasks for This Phase (beyond pages)

1. **Internal link audit** — every page should have 3+ outbound internal links. Fix orphans.
2. **Schema validation** — run all pages through Google Rich Results Test and fix any warnings.
3. **Core Web Vitals check** — use PageSpeed Insights on 5 representative pages. LCP target <2.5s, CLS <0.1, INP <200ms.
4. **Sitemap pruning** — if any Phase 1-5 pages have zero impressions after 6 months, consider noindex + removing from sitemap to save crawl budget.
5. **Alt text audit** — every `<img>` on the entire site.
6. **Featured-snippet optimization** — find the 10 glossary pages with the most impressions, ensure each has a direct-answer paragraph in the first 200 words.
